# Regex สำหรับตรวจมาตรฐานไวยากรณ์ภาษาไทย
## ข้อความ
กาแฟร้อนๆ มีกลิ่นหอมฟุ้งไปทั่วห้อง เด็ก ๆ กล้อนผมก็วิ่งเล่นอย่างสนุกสนาน เพียงผู้หญิงคนนั้นเดินผ่านไปอย่างรวดเร็ว ใบไม้ร่วงหล่นลงมาเป็นจำนวนมาก ฉันชอบกินผลไม้สดๆ โดยเฉพาะแตงโม สมุทรสาครเป็นจังหวัดที่อยู่ใกล้กรุงเทพฯ เขาแต่งกลอนแล้วพูดว่า "ไม่เป็นไร" แล้วยิ้มให้อย่างแกล้วกล้า นักเรียนทุกคนต้องใส่เครื่องแบบ ดอกไม้บานสะพรั่งในสวน ฤดูฝนปีนี้มีน้ำมาก ญาติๆ มาเยี่ยมเยือนบ้านเป็นประจำ ธรรมชาติสวยงามมาก ฌานสมาธิช่วยให้จิตใจสงบ ปลาทูทอดกรอบอร่อยมาก แม่ซื้อของมาเต็มตะกร้า พ่อขับรถไปทำงานทุกวัน ไก่ตัวนั้นขันดังมาก เธอใจดีมากๆ เลย ชาวบ้านชุมชนไม่สนใจความละเอียดละออ เคล้าความงมงาย มะขามเปรี้ยวมะเขือหวานอร่อยมาก อยากตื่นเต้นดีใจจริงๆ ถ้าเทียบความดั้งเดิมทำไมคนแซ่ตั้งดี เขาว่าอย่าอยู่อย่างอยาก อย่ามาชวนทะเลาะได้ไหม เอาอะไรมาอีกเนาะโมเมไป แบบว่าอำนาจใหญ่โต แบบนี้นั่นนี่โน่น กล้าเกล้ามาดู เงื่อนไขเหล่านี้ เรื่องราวเรื่อยเปื่อย เสื้อผ้าสวยงาม เหมือนเดิมทุกอย่าง คือแกงเผ็ดแกงจืด ใจเย็นใจร้อน เตียงนอนเตาไฟ เกลือป่นพริกไทย เตาไฟลั่นดังเปรี๊ยะ ๆ เปลือกหอยกระเทาะออกมา เหาะเหินเดินอากาศ โบยบินไปในอากาศทุกอย่างโบ๋เบ๋ มาเราจะไปเที่ยวหัวหิน เมากันให้เละเทะและแทะไปเลย สักครู่นึงก็ได้เนื้อหาตุ๊ดตู่ปู่บู้บี้ละ ดีไม่ดีก็ลองดู ก็โน๊ะ หาโอกาสคุยกันหน่อยกับพวกเกรียนคีย์บอร์ด ลำนำลำน้ำแคว เทพีเพรียงเดินเพลิดเพลินเหลือคนาเพียงใจ ดับเบิ้ลเปิ้น

กาาแฟร้้อนๆ มีกลิ่ินหอมฟุ้งไปทั่ัวห้อง เด็็ก ๆ กล้อนผมก่็วิิ่งเล่่นอย่างสนุ่กสนาน เพีผู้หญิิงคนนัันเดินผ่านไปอย่างรวดเร็ว ใบไมัร่วงหล่นลงมาเป็นจำนวนมาก ฉันชอบกินผลไม้สดๆ โโดยเฉพาะแตงโม สมุทรสาครเป็นจังหวัดที่อยู่ใกล้กรุงเทพฯฯ เขาแต่งกลอนพูดว่า "ไม่เป็ไร" แล้วยิ้้มให้อย่งแกล้วกล้า นักเรียนทุกคนต้องใใส่เครื่องแบบ ดอกไม้บานสะพรั่งในืสวน ฤดูฝนปีนี้มีน้ำมาก ญาติๆ มาเยี่อมเยืยนบ้านเป็นประจำ ธรรมชาติสวยงามมาก ฌานสมาธิช่วยให้จิตใจสงบ ปลาทูทอดกรอบอร่่อยมาก แม่ซื้อของมาเต็่มตะกร้า พ่อขับรถไปทำงานทุูกวัน ไก่ตัวนั้นขันดังมาก เธอใจดีมาาากๆๆ เลย ชาวบ้านชุุมชนไไม่สนใจความละเอียดละออ เคล้้าความงมงาย มะะขามเปร้ียวมะเขือหวานอร่อยมาก อยากตื่นเต้้นดีใจจริงๆ ถ้าเทียบความดั้งเดิมเทำไมคนแซ่ตั้งดี เขาว่าอย่าอยู่อย่างอยาก อย่ามาชวนทะเลาะได้ไหม เอาอะไรมาอีกเนาะโมเมีไป แบบว่าอ่ำนาจใหญ่โต แบบนี้นั่นนี่โนีน กล้าเกล้ามาดู เงื่อนไขเหล่านี้ เรื่องราวเรื่อยเปื่อย เสื้อผ้าสวยงาม เหมือนเดิมทุกอย่าง คืแกงเผ็ดแกงจืด ใจเย็นใจร้อน เตียงนอนเตาไฟ เกลือป่นพริกไทย เตาไฟลั่นดังเปรี๊ยะ ๆ เปลือกหอยกระทาะออกมา เห็าะเหินเดินอากาศ โบ็ยบินไปในอากาศทุกอย่างโบ๋๋เบ๋่ มาเราจะไปเที่ยวหัวหิน เมากันให้เละเทะและแทะไปเลย สักครู่นึงก็ได้เนื้อหาตุ๊ดตู่ปุีบู้บี้ละ ดีไม่ดีก็ลองดู ก็่โน๊าะ หาโอาคุ้ยกัันหน่อย กับพวกเกรียนคีย์์บอร์ด  แลำนีำลำน้ำแคว เทพีเพรียงเดิเพลิดเพลินเหลืคนาเพีใจ ดับเบิ้ลเปิ้น

## ตามกฏภาษาไทยเบื้องต้น

### ตรวจจับไม้หันอากศที่ไม่มีพยัญชนะ
```
(?<![ก-ฮ])ั|ั(?![ก-ฮ]|[่้๊๋][ก-ฮ])
```
จับในกรณีที่
- ไม้หันอากาศไม่มีพยัญชนะ
- ไม้หันอากาศที่ไม่มีตัวสะกด
### ตรวจจับวรรณยุกต์ที่ไม่มีพยัญชนะ
```
(?<![ก-ฮ][ิีืึุูั]?)[่้๊๋]
```
จับในกรณีที่
- วรรณยุกต์ไม่มีพยัญชนะนำหน้า
- วรรณยุกต์ที่ถ้ามีสระบนล่างไม่อยู่ก่อนวรรณยุกต์
### ตรวจจับสระหน้าที่ไม่มีพยัญชนะตามท้าย
```
[เแไโใ](?![ก-ฮ])|[แไโใ][ก-ฮ][ิีึืุูำั็า]
```
จับในกรณีที่
- สระหน้าไม่ตามด้วยพยัญชนะ
- สระหน้าตามด้วยพยัญชนะที่มีสระอื่นกำกับรวมทั้งไม้หันอากาศ
### ตรวจจับสระบนล่างและหลังที่ไม่มีพยัญชนะนำ
```
(?<![ก-ฮ][่้๊๋]?)[ิีึืุูำำ]
```
จับในกรณีที่
- สระบนไม่มีพยัญชนะข้างล่างก่อน
- สระล่างไม่มีพยัญชนะนำหน้า
- สระอำไม่มีพยัญชนะนำหน้า
### ตรวจจับการใช้สระอะที่เป็นสระอะและสระเอาะโอะเออะเอะแอะที่ไม่มีพยัญชนะข้างหน้า
```
(?<!เ([ก-ฮ]{1,2}[่้๊๋]?)า|เ[ก-ฮ]อ|โ[ก-ฮ]|เ[ก-ฮ]|แ[ก-ฮ]|[ก-ฮ][่้๊๋]?)ะ
```
จับในกระณีที่
- สระอะไม่มีพยัญชนะนำหน้า
- สระอะและข้างหน้าไม่ใช่รูปแบบสระเอาะ
- สระอะและข้างหน้าไม่ใช่รูปแบบสระโอะ
- สระอะและข้างหน้าไม่ใช่รูปแบบสระเออะ
- สระอะและข้างหน้าไม่ใช่รูปแบบสระแอะ
- สระอะและข้างหน้าไม่ใช่รูปแบบสระเอะ
### ตรวจจับการใช้สระอาที่เป็นสระอาและสระเอาที่ไม่มีพยัญชนะข้างหน้า
```
(?<!เ[ก-ฮ]{1,2}[่้๊๋]?|[ก-ฮ][่้๊๋]?)า
```
จับในกรณีที่
- สระอาไม่มีพยัญชนะนำ
- สระอาไม่อยู่ในรูปแบบสระเอา

### สระเอสระเอียและสระเอือและเออเปลี่ยนรูปและสระเอะเปลี่ยนรูป
```
เ(?![ก-ฮ])|เ[ก-ฮ]ี(?![่้๊๋]?ย[ก-ฮ]?)|เ[ก-ฮ]{1,2}ื(?![่้๊๋]?อ[ก-ฮ]?)|เ[ก-ฮ]{1,2}ิ(?![่้๊๋]?[ก-ฮ])|เ[ก-ฮ]{1,2}็(?![ก-ฮ])
```
จับในกรณีที่
- สระเอแล้วไม่มีพยัญชนะตาม
- สระเอแล้วไม่ใช่สระเอีย
- สระเอแล้วไม่ใช่สระเอือ
- สระเออเปลี่ยนรูป(แบบมีตัวสะกด)แต่ขาดตัวสะกด
- สระเอะเปลี่ยนรูป(ให้ไม้ไต่คู้แบบมีตัวสะกด)แต่ขาดตัวสะกด

### สระอือต้องมีตัวสะกด
```
[ื](?![่้๊๋]?[ก-ฮ])
```
- จับสระอืที่ไม่มีตัวสะกด

### สระอำ
```
([แเโไใ][ก-ฮ]|[ก-ฮ][ิีึืุูาะ])ำ
```
จับในกรณีที่
- พยัญชนะที่มีสระอำมีสระอื่นร่วมอยู่ด้วย
### ตรวจจับสระหน้าแล้วตามด้วยสระหลังบนล่างทันที
```
[เแโไใ][ิีึืุูั็าะำ]
```
จับในกรณีที่
- สระหน้าแล้วตามด้วยสระบนล่างหลังทันที

## ตรวจสระซ้ำซ้อน
### สระหน้าซ้ำซ้อน
```
[เแโไใ]{2,}
```
จับในกรณีที่
- สระหน้าซ้ำซ้อน
### สระบนล่างและไม้หันอากาศซ้ำซ้อน
```
[ิีึืูุั็]{2,}
```
จับในกรณีที่สระบนล่างซ้ำซ้อนไม้หันอากาศไม้ไต่คู้ซ้ำซ้อน
### วรรณยุกต์ซ้ำซ้อน
```
[่้๊๋]{2,}
```
จับกรณีที่วรรณยุกต์ซ้ำซ้อน
### ทัณฑฆาตซ้ำซ้อน
```
[์]{2,}
```
จับกรณีที่ทัณฑฆาตซ้ำซ้อน
## ตรวจลำดับ
### ลำดับสระบนและวรรณยุกต์
```
[่้๊๋][ิีึืุูั็]
```
จับกรณีที่วรรณยุกต์มาก่อนสระบนล่าง

## ตรวจนับคำพิเศษ
### อย่าอยู่อย่างอยาก
```
((?<!((?<![เแไโใ])[ก-ฮ]{1,2}[่้๊๋]?)|[ก-ฮ]ื[่้๊๋]?)อย)(?!(่า)|(ู่)|(่าง)|(าก))
```
จับกรณีที่
- "อย" ไม่ใช่ส่วนหนึ่งของสระออและสระเอือแต่ไม่ใช่ "อย่า" "อยู่" "อย่าง" "อยาก"
