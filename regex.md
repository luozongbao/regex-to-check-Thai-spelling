# Regex สำหรับตรวจการสะกดสระและวรรณยุกต์ภาษาไทย
## ข้อความ
กาแฟร้อนๆ มีกลิ่นหอมฟุ้งไปทั่วห้อง เด็ก ๆ กล้อนผมก็วิ่งเล่นอย่างสนุกสนาน เพียงผู้หญิงคนนั้นกัดผมเดินผ่านไปอย่างรวดเร็ว ใบไม้ร่วงหล่นลงมาเป็นจำนวนมาก ฉันชอบกินผลไม้สดๆ ทำให้ร่างกายแข็งแรง โดยเฉพาะแตงโม สมุทรสาครเป็นจังหวัดที่อยู่ใกล้กรุงเทพฯ เขาแต่งกลอนแล้วพูดว่า "ไม่เป็นไร" แล้วยิ้มให้อย่างแกล้วกล้า นักเรียนทุกคนต้องใส่เครื่องแบบ ดอกไม้บานสะพรั่งในสวน ฤดูฝนปีนี้มีน้ำมาก ญาติๆ มาเยี่ยมเยือนบ้านเป็นประจำ ธรรมชาติสวยงามมาก ฌานสมาธิช่วยให้จิตใจสงบ ปลาทูทอดกรอบอร่อยมาก แม่ซื้อของมาเต็มตะกร้า พ่อขับรถไปทำงานทุกวัน ไก่ตัวนั้นขันดังมาก เธอใจดีมากๆ เลย ชาวบ้านชุมชนไม่สนใจความละเอียดละออ เคล้าความงมงาย มะขามเปรี้ยวมะเขือหวานอร่อยมาก อยากตื่นเต้นดีใจจริงๆ ถ้าเทียบความดั้งเดิมทำไมคนแซ่ตั้งดี เขาว่าอย่าอยู่อย่างอยาก อย่ามาชวนทะเลาะได้ไหม เอาอะไรมาอีกเนาะโมเมไป แบบว่าอำนาจใหญ่โต แบบนี้นั่นนี่โน่น กล้าเกล้ามาดู เงื่อนไขเหล่านี้ เรื่องราวเรื่อยเปื่อย เสื้อผ้าสวยงาม เหมือนเดิมทุกอย่าง คือแกงเผ็ดแกงจืด ใจเย็นใจร้อน เตียงนอนเตาไฟ เกลือป่นพริกไทย เตาไฟลั่นดังเปรี๊ยะ ๆ เปลือกหอยกระเทาะออกมา เหาะเหินเดินอากาศ โบยบินไปในอากาศทุกอย่างโบ๋เบ๋ มาเราจะไปเที่ยวหัวหิน เมากันให้เละเทะและแทะไปเลย สักครู่นึงก็ได้เนื้อหาตุ๊ดตู่ปู่บู้บี้ละ ดีไม่ดีก็ลองดู ก็โน๊ะ หาโอกาสคุยกันหน่อยกับพวกเกรียนคีย์บอร์ด ลำนำลำน้ำแคว เทพีเพรียงเดินเพลิดเพลินเหลือคนาเพียงใจ ดับเบิ้ลเปิ้น กล้องถ่ายกล้วยล้มส้มใหญ่ไปไม่ได้ใกล้ ๆ หรือไกลก็ได้ ลองดูคำประหลาดๆ หน่อย เช่น แข็งทื่อ โกโก้ เกล้อะ โกล้ะ เกละ แกละ กละ กลาก เกลื้อน กมลสันดารเป็นไปเพื่อความสนุกสนาน และใช้บัตรทองความศักดิ์สิทธิ สวาร์มซี๊ดส์ซัดส์คอมพิวเตอร์สวมสเกิร์ทนอนเกร็งเฉิ่ม เปิดสวิทช์มีเซ็กซ์ในสตาร์ทอร์ชเสปิร์มยิ้มแฉ่ง หลายกิโลวัตต์ ถูกแฮ็กซ์คอนเฟิร์ม คอมพิวเตอร์ กราฟ สเกิร์ท ชีส ซัดส์ ดัชท์ สวิทช์ เซ็กซ์ วัตต์ เสปิร์ม สตาร์ แบลคลิสต์ ไวท์ลิสต์ ฟีสต์

กาาแฟร้้อนๆ มีกลิ่ินหอมฟุ้งไปทั่ัวห้อง เด็็ก ๆ กล้อนผมก่็วิิ่งเล่่นอย่างสนุ่กสนาน เพีผู้หญิิงคนนัันกัผมเดินผ่านไปอย่างรวเร็ว ใบไมัร่วงหล่นลงมาเป็นจำนวนมาก ฉันชอบกินผลไม้สดๆ ทำให้ร่างกายแข็แรง โโดยเฉพาะแตงโม สมุทรสาครเป็นจังหวัดที่อยู่ใกล้กรุงเทพฯฯ เขาแต่งกลอนพูดว่า "ไม่เป็ไร" แล้วยิ้้มให้อย่งแกล้วกล้า นักเรียนทุกคนต้องใใส่เครื่องแบบ ดอกไม้บานสะพรั่งในืสวน ฤดูฝนปีนี้มีน้ำมาก ญาติๆ มาเยี่อมเยืยนบ้านเป็นประจำ ธรรมชาติสวงามมาก ฌานสมาธิช่วยให้จิตใจสงบ ปลาทูทอดกรอบอร่่อยมาก แม่ซื้อของมาเต็่มตะกร้า พ่อขัรถไปทำงานทุูกวัน ไก่ตัวนั้นขันดังมาก เธอใจดีมาาากๆๆ เลย ชาวบ้านชุุมชนไไม่สนใจความละเอียดละออ เคล้้าความงมงาย มะะขามเปร้ียวมะเขือหวานอร่อยมาก อยากตื่นเต้้นดีใจจริงๆ ถ้าเทียบความดั้งเดิมเทำไมคนแซ่ตั้งดี เขาว่าอย่าอยู่อย่างอยาก อย่ามาชวนทะเลาะได้ไหม เอาอะไรมาอีกเนาะโมเมีไป แบบว่าอ่ำนาจใหญ่โต แบบนี้นั่นนี่โนีน กล้าเกล้ามาดู เงื่อนไขเหล่านี้ เรื่องราวเรื่อยเปื่อย เสื้อผ้าสวยงาม เหมือนเดิมทุกอย่าง คืแกงเผ็ดแกงจืด ใจเย็นใจร้อน เตียงนอนเตาไฟ เกลือป่นพริกไทย เตาไฟลั่นดังเปรี๊ยะ ๆ เปลือกหอยกระทาะออกมา เห็าะเหินเดินอากาศ โบ็ยบินไปในอากาศทุกอย่างโบ๋๋เบ๋่ มาเราจะไปเที่ยวหัวหิน เมากันให้เละเทะและแทะไปเลย สักครู่นึงก็ได้เนื้อหาตุ๊ดตู่ปุีบู้บี้ละ ดีไม่ดีก็ลองดู ก็่โน๊าะ หาโอาคุ้ยกัันหน่อย กับพวกเกรียนคีย์์บอร์ด  แลำนีำลำน้ำแคว เทพีเพรียงเดิเพลิดเพลินเหลืคนาเพีใจ ดับเบิ้ลเปิ้น กล้ถ่ายกล้ล้มส้มหญ่ไปไม่ได้ใกล้ ๆ หรือกลก็ได้ ลองดูคำประหลาดๆ หน่อย เช่น เขึงแทื่อ โกาโก์ เกล้อะ โกล้ะ เกละ แกละ กละ กลาก เกลื้อน กมลสัดารเป็นไปเพื่อความสนุกสนาน และใช้บัตรทองความศัดิ์สิทธิ์ สวาร์มซืส์ซัส์คอมพิวเตอร์สวมสเกิร์ทนอนเกร็งเฉิ่ม เปิดสวิช์มีเซ็ซ์ในสตาร์ทอร์ชเสปิร์ยิ้มแฉ่ง หลายกิโลวัต์ ถูกแฮ็ซ์คอนเฟิร์ คอมพิวเตอร์ กราฟ สเกิร์ท ชีส ซัดส์ ดัชท์ สวิช์ เซ็ซ์ วัตต์ เสปิร์ สตาร์ แบลคลิสต์ ไวท์ลิสต์ ฟีต์

## Regex ตามกฏภาษาไทยเบื้องต้น

## ตรวจสระซ้ำซ้อน
### สระหน้าซ้ำซ้อน
```
[เแโไใ]{2,}
```
จับในกรณีที่
- สระหน้าซ้ำซ้อน
### สระบนล่างและไม้หันอากาศซ้ำซ้อน
```
[ิีึืูุั็]{2,}
```
จับในกรณีที่สระบนล่างซ้ำซ้อนไม้หันอากาศไม้ไต่คู้ซ้ำซ้อน
### วรรณยุกต์ซ้ำซ้อน
```
[่้๊๋]{2,}
```
จับกรณีที่วรรณยุกต์ซ้ำซ้อน
### ทัณฑฆาตซ้ำซ้อน
```
[์]{2,}
```
จับกรณีที่ทัณฑฆาตซ้ำซ้อน

### ลำดับสระบนและวรรณยุกต์
```
[่้๊๋][ิีึืุูั็]
```
จับกรณีที่วรรณยุกต์มาก่อนสระบนล่าง

### ตรวจจับสระหน้าแล้วตามด้วยสระหลังบนล่างทันที
```
[เแโไใ][ิีึืุูั็าะำ]
```
จับในกรณีที่
- สระหน้าแล้วตามด้วยสระบนล่างหลังทันที

### ตรวจจับไม้หันอากศที่ไม่มีพยัญชนะ (สระอะเปลี่ยนรูป) และตัวสะกด
```
(?<![ก-ฮ])(?:ั)
```
จับในกรณีที่
- ไม้หันอากาศไม่มีพยัญชนะ

### ตรวจจับไม้หันอากศ (สระอะเปลี่ยนรูป) ที่ไม่มีตัวสะกด
```
(?:ั)(?![่้๊๋]?(?:(?![ผฝหอฮฉฌรฃฅ])[ก-ฮ]))
```
- ไม้หันอากาศที่ไม่มีตัวสะกด

### ตรวจจับวรรณยุกต์ที่ไม่มีพยัญชนะ
```
(?<![ก-ฮ][ิีืึุูั]?)[่้๊๋]
```
จับในกรณีที่
- วรรณยุกต์ไม่มีพยัญชนะนำหน้า
- วรรณยุกต์ที่ถ้ามีสระบนล่างไม่อยู่ก่อนวรรณยุกต์

### ตรวจจับสระหน้าที่ไม่มีพยัญชนะตามท้าย
```
[เแไโใ](?![ก-ฮ])
```
จับในกรณีที่
- สระหน้าไม่ตามด้วยพยัญชนะ

### ตรวจจับสระนำหน้าที่ไม่ควรสระอื่น
```
[ไโใ][ก-ฮ][่้๊๋]?[ิีึืุูำั็า์]|แ[ก-ฮ][่้๊๋]?[ีืึิุูำัา์]|เ[ก-ฮ][่้๊๋]?[ึุูำั์]
```
- สระไอโอตามด้วยพยัญชนะที่มีสระอิอีอึอือุอูอำอาไม้หันอากาศไม้ไต่คู้การันต์
- สระแอตามด้วยพยัญชนะที่ไม่มีสระอิอีอึอืออุอูอำอาไม้หันอากาศการันต์
- สระเอตามด้วยพยัญชนะที่ไม่มีสระอึอุอูอำไม้หันอากาศการันต์


### สระแอะลดรูป
```
แ[ก-ฮ]็(?![ก-ฮ])
```
จับในกรณีที่ 
- สระแอะไม่มีตัวสะก็ด

### ตรวจจับสระบนล่างและหลังที่ไม่มีพยัญชนะนำ
```
(?<![ก-ฮ][่้๊๋]?)[ิีึืุูำำ]
```
จับในกรณีที่
- สระบนไม่มีพยัญชนะข้างล่างก่อน
- สระล่างไม่มีพยัญชนะนำหน้า
- สระอำไม่มีพยัญชนะนำหน้า
### ตรวจจับการใช้สระอะที่เป็นสระอะและสระเอาะโอะเออะเอะแอะที่ไม่มีพยัญชนะข้างหน้า
```
(?<!เ[ก-ฮ]{1,2}[่้๊๋]?[าอ]|[โเแ]?[ก-ฮ]{1,2}[่้๊๋]?)ะ
```
จับในกระณีที่
- สระอะไม่มีพยัญชนะนำหน้า
- สระอะและข้างหน้าไม่ใช่รูปแบบสระเอาะ
- สระอะและข้างหน้าไม่ใช่รูปแบบสระโอะ
- สระอะและข้างหน้าไม่ใช่รูปแบบสระเออะ
- สระอะและข้างหน้าไม่ใช่รูปแบบสระแอะ
- สระอะและข้างหน้าไม่ใช่รูปแบบสระเอะ
### ตรวจจับการใช้สระอาที่เป็นสระอาและสระเอาที่ไม่มีพยัญชนะข้างหน้า
```
(?<!เ?[ก-ฮ]{1,2}[่้๊๋]?)า
```
จับในกรณีที่
- สระอาไม่มีพยัญชนะนำ
- สระอาไม่อยู่ในรูปแบบสระเอา

### สระเอสระเอียและสระเอือ
```
เ(?![ก-ฮ])|เ[ก-ฮ](?:ี(?![่้๊๋]?ย[ก-ฮ]?)|ื(?![่้๊๋]?อ[ก-ฮ]?))
```
จับในกรณีที่
- สระเอแล้วไม่มีพยัญชนะตาม
- สระเอแล้วไม่ใช่สระเอีย
- สระเอแล้วไม่ใช่สระเอือ

### สระเออเปลี่ยนรูป (เ-ิม) และสระเอะเปลี่ยนรูป(เ-็ม)
```
เ[ก-ฮ]{1,2}(?:ิ(?![่้๊๋]?[ก-ฮ](?![ะาิีึืุูำั็่้๊๋]))|็(?![ก-ฮ](?![ะาิีึืุูำั็่้๊๋])))
```
จับกรณีที่
- สระเออเปลี่ยนรูป(แบบมีตัวสะกด)แต่ขาดตัวสะกด
- สระเอะเปลี่ยนรูป(ให้ไม้ไต่คู้แบบมีตัวสะกด)แต่ขาดตัวสะกด

### สระอือต้องมีตัวสะกด
```
[ื](?![่้๊๋]?[ก-ฮ](?![ะาิีึืุูำั็่้๊๋]))
```
- จับสระอืที่ไม่มีตัวสะกด

### สระอำ
```
(?:[แเโไใ][ก-ฮ]|[ก-ฮ][ิีึืุูาะ])ำ
```
จับในกรณีที่
- พยัญชนะที่มีสระอำมีสระอื่นร่วมอยู่ด้วย

## วิเคราะห์ตามกฏภาษาไทยเพิ่มเติม

**ตัวการันต์กับเครื่องหมายฑัณทฆาตบังคับต้องมีตัวสะกดตาม**

ไม้หันอากาศตามด้วยตัวสะกดทันทีจึงตามด้วยตัวการันต์
สระเออเปลี่ยนรูปต้องใส่ตัวสะกดหลังตัวการันต์
ไม้หันอากาศตามด้วยตัวสะกดทันทีจึงตามด้วยตัวการันต์ (ไม่บังคับมีตัวการันต์ข้างหลัง)
สระออต้องใส่ตัวการันต์ก่อนแล้วตามด้วยตัวสะกด(ไม่บังคับต้องมีตัวสะกด)
สระอี "ไม่บังคับ" ตัวสะกด์แต่ต้องใส่การันต์หลังตัวสะกด
