# Regex สำหรับตรวจการสะกดสระและวรรณยุกต์ภาษาไทย
## ข้อความ
กาแฟร้อนๆ มีกลิ่นหอมฟุ้งไปทั่วห้อง เด็ก ๆ กล้อนผมก็วิ่งเล่นอย่างสนุกสนาน เพียงผู้หญิงคนนั้นกัดผมเดินผ่านไปอย่างรวดเร็ว ใบไม้ร่วงหล่นลงมาเป็นจำนวนมาก ฉันชอบกินผลไม้สดๆ ทำให้ร่างกายแข็งแรง โดยเฉพาะแตงโม สมุทรสาครเป็นจังหวัดที่อยู่ใกล้กรุงเทพฯ เขาแต่งกลอนแล้วพูดว่า "ไม่เป็นไร" แล้วยิ้มให้อย่างแกล้วกล้า นักเรียนทุกคนต้องใส่เครื่องแบบ ดอกไม้บานสะพรั่งในสวน ฤดูฝนปีนี้มีน้ำมาก ญาติๆ มาเยี่ยมเยือนบ้านเป็นประจำ ธรรมชาติสวยงามมาก ฌานสมาธิช่วยให้จิตใจสงบ ปลาทูทอดกรอบอร่อยมาก แม่ซื้อของมาเต็มตะกร้า พ่อขับรถไปทำงานทุกวัน ไก่ตัวนั้นขันดังมาก เธอใจดีมากๆ เลย ชาวบ้านชุมชนไม่สนใจความละเอียดละออ เคล้าความงมงาย มะขามเปรี้ยวมะเขือหวานอร่อยมาก อยากตื่นเต้นดีใจจริงๆ ถ้าเทียบความดั้งเดิมทำไมคนแซ่ตั้งดี เขาว่าอย่าอยู่อย่างอยาก อย่ามาชวนทะเลาะได้ไหม เอาอะไรมาอีกเนาะโมเมไป แบบว่าอำนาจใหญ่โต แบบนี้นั่นนี่โน่น กล้าเกล้ามาดู เงื่อนไขเหล่านี้ เรื่องราวเรื่อยเปื่อย เสื้อผ้าสวยงาม เหมือนเดิมทุกอย่าง คือแกงเผ็ดแกงจืด ใจเย็นใจร้อน เตียงนอนเตาไฟ เกลือป่นพริกไทย เตาไฟลั่นดังเปรี๊ยะ ๆ เปลือกหอยกระเทาะออกมา เหาะเหินเดินอากาศ โบยบินไปในอากาศทุกอย่างโบ๋เบ๋ มาเราจะไปเที่ยวหัวหิน เมากันให้เละเทะและแทะไปเลย สักครู่นึงก็ได้เนื้อหาตุ๊ดตู่ปู่บู้บี้ละ ดีไม่ดีก็ลองดู ก็โน๊ะ หาโอกาสคุยกันหน่อยกับพวกเกรียนคีย์บอร์ด สัตว์ที่อาศัยลำนำลำน้ำแคว เทพีเพรียงเดินเพลิดเพลินเหลือคนาเพียงใจ ดับเบิ้ลเปิ้น กล้องถ่ายกล้วยล้มส้มใหญ่ไปไม่ได้ใกล้ ๆ หรือไกลก็ได้ ลองดูคำประหลาดๆ หน่อย เช่น แข็งทื่อ โกโก้ เกล้อะ โกล้ะ เกละ แกละ กละ กลาก เกลื้อน กมลสันดารเป็นไปเพื่อความสนุกสนาน และใช้บัตรทองความศักดิ์สิทธิ สวาร์มซี๊ดส์ซัดส์คอมพิวเตอร์สวมสเกิร์ทนอนเกร็งเฉิ่ม เปิดสวิทช์มีเซ็กซ์ในสตาร์ทอร์ชเสปิร์มยิ้มแฉ่ง หลายกิโลวัตต์ ถูกแฮ็กซ์คอนเฟิร์ม คอมพิวเตอร์ กราฟ สเกิร์ท ชีส ซัดส์ ดัชท์ สวิทช์ เซ็กซ์ วัตต์ เสปิร์ม สตาร์ แบลคลิสต์ ไวท์ลิสต์ ฟีสต์ เธอยอมร่วมมือป่าว โอ้ยๆ โอดโอยเอย ยานอวกาศพุ่งทะยานว่องไว ผล็อยหลับ ชื่อย่อมสะท้อนประสบการณ์ มังกรวสันต์ สิบวสันต์ ร้อยวสันต์ อวมงคล นวเวทย์ วนอุทยาน ขวนขวาย รวบรวม จวนตัวสัตว์แพทย์ ชู่ แองเจลิกา มาร์เกริตา ทะเลก็ทำให้เกิดคลื่นสูง หน้าที่ของวณิพก ไม่ว่าจะแลกด้วยอะไรก็ตาม

กาาแฟร้้อนๆ มีกลิ่ินหอมฟุ้งไปทั่ัวห้อง เด็็ก ๆ กล้อนผมก่็วิิ่งเล่่นอย่างสนุ่กสนาน เพีผู้หญิิงคนนัันกัผมเดินผ่านไปอย่างรวเร็ว ใบไมัร่วงหล่นลงมาเป็นจำนวนมาก ฉันชอบกินผลไม้สดๆ ทำให้ร่างกายแข็แรง โโดยเฉพาะแตงโม สมุทรสาครเป็นจังหวัดที่อยู่ใกล้กรุงเทพฯฯ เขาแต่งกลอนพูดว่า "ไม่เป็ไร" แล้วยิ้้มให้อย่งแกล้วกล้า นักเรียนทุกคนต้องใใส่เครื่องแบบ ดอกไม้บานสะพรั่งในืสวน ฤดูฝนปีนี้มีน้ำมาก ญาติๆ มาเยี่อมเยืยนบ้านเป็นประจำ ธรรมชาติสวงามมาก ฌานสมาธิช่วยให้จิตใจสงบ ปลาทูทอดกรอบอร่่อยมาก แม่ซื้อของมาเต็่มตะกร้า พ่อขัรถไปทำงานทุูกวัน ไก่ตัวนั้นขันดังมาก เธอใจดีมาาาาาาาาากๆๆ เลย ชาวบ้านชุุมชนไไม่สนใจความละเอียดละออ เคล้้าความงมงาย มะะขามเปร้ียวมะเขือหวานอร่อยมาก อยากตื่นเต้้นดีใจจริงๆ ถ้าเทียบความดั้งเดิมเทำไมคนแซ่ตั้งดี เขาว่าอย่าอยู่อย่างอยาก อย่ามาชวนทะเลาะได้ไหม เอาอะไรมาอีกเนาะโมเมีไป แบบว่าอ่ำนาจใหญ่โต แบบนี้นั่นนี่โนีน กล้าเกล้ามาดู เงื่อนไขเหล่านี้ เรื่องราวเรื่อยเปื่อย เสื้อผ้าสวยงาม เหมือนเดิมทุกอย่าง คืแกงเผ็ดแกงจืด ใจเย็นใจร้อน เตียงนอนเตาไฟ เกลือป่นพริกไทย เตาไฟลั่นดังเปรี๊ยะ ๆ เปลือกหอยกระทาะออกมา เห็าะเหินเดินอากาศ โบ็ยบินไปในอากาศทุกอย่างโบ๋๋เบ๋่ มาเราจะไปเที่ยวหัวหิน เมากันให้เละเทะและแทะไปเลย สักครู่นึงก็ได้เนื้อหาตุ๊ดตู่ปุีบู้บี้ละ ดีไม่ดีก็ลองดู ก็่โน๊าะ หาโอาคุ้ยกัันหน่อย กับพวกเกรียนคีย์์บอร์ด  สัตว์ที่อาศัยแลำนีำลำน้ำแคว เทพีเพรียงเดิเพลิดเพลินเหลืคนาเพีใจ ดับเบิ้ลเปิ้น กล้ถ่ายกล้ล้มส้มหญ่ไปไม่ได้ใกล้ ๆ หรือกลก็ได้ ลองดูคำประหลาดๆ หน่อย เช่น เขึงแทื่อ โกาโก์ เกล้อะ โกล้ะ เกละ แกละ กละ กลาก เกลื้อน กมลสัดารเป็นไปเพื่อความสนุกสนาน และใช้บัตรทองความศัดิ์สิทธิ์ สวาร์มซืส์ซัส์คอมพิวเตอร์สวมสเกิร์ทนอนเกร็งเฉิ่ม เปิดสวิช์มีเซ็ซ์ในสตาร์ทอร์ชเสปิร์ยิ้มแฉ่ง หลายกิโลวัต์ ถูกแฮ็ซ์คอนเฟิร์ คอมพิวเตอร์ กราฟ สเกิร์ท ชีส ซัดส์ ดัชท์ สวิช์ เซ็ซ์ วัตต์ เสปิร์ สตาร์ แบลคลิสต์ ไวท์ลิสต์ ฟีต์ เธอยอมร่วมมือป่าว โอ้ยๆ โอดโอยเอย ยานอวกาศพุ่งทะยานว่องไว ผล็อยหลับ ชื่อย่อมสะท้อนประสบการณ์ มังกรวสันต์ สิวสันต์ ร้อวสันต์ อวมงคล นวเวทย์ วนอุทยาน ขวนขวาย รวบรวม จวนตัวสัตวแพทย์ ชู่วววววววววววว แองเจลิกา มาร์เกริตา ทะเลก็ทำให้เกิดคลื่นสูง หน้าที่ของวณิพก ไม่ว่าจะแลกด้วยอะไรก็ตาม

## Regex ตามกฏภาษาไทยเบื้องต้น

## ตรวจสระซ้ำซ้อน
### สระหน้าซ้ำซ้อน
```
[เแโไใ]{2,}
```
***จับกรณีที่***
- สระหน้าซ้ำซ้อน
### สระบนล่างและไม้หันอากาศซ้ำซ้อน
```
[ิีึืูุั็]{2,}
```
***จับกรณีที่***
สระบนล่างซ้ำซ้อนไม้หันอากาศไม้ไต่คู้ซ้ำซ้อน
### วรรณยุกต์ซ้ำซ้อน
```
[่้๊๋]{2,}
```
***จับกรณีที่***
วรรณยุกต์ซ้ำซ้อน
### ทัณฑฆาตซ้ำซ้อน
```
[์]{2,}
```
***จับกรณีที่***
ทัณฑฆาตซ้ำซ้อน

### ลำดับสระบนและวรรณยุกต์
```
[่้๊๋][ิีึืุูั็]
```
***จับกรณีที่***
วรรณยุกต์มาก่อนสระบนล่าง

### ตรวจจับสระบนล่างและหลังที่ไม่มีพยัญชนะนำ
```
(?<![ก-ฮ][่้๊๋]?)[ิีึืุูำ็ั]
```
***จับกรณีที่***
- สระบนไม่มีพยัญชนะข้างล่างก่อน รวมไม้ไต่คู้และไม้หันอากาศ
- สระล่างไม่มีพยัญชนะนำหน้า
- สระหลังนับเพียงสระอำไม่มีพยัญชนะนำหน้า (สระอะกับสระอาแยกต่างหาก)

### ตรวจจับสระหน้าแล้วตามด้วยสระหลังบนล่างทันที
```
[เแโไใ][ิีึืุูั็าะำ]
```
***จับกรณีที่***
- สระหน้าแล้วตามด้วยสระบนล่างหลังทันที

### ทัณฑฆาตที่ไม่มีพยัญชนะ
```
(?<![ก-ฮ][ิุ]?)(?:์)
```
***จับกรณีที่***
- ทัณฑฆาตไม่มีพยัญชนะ อาจมีสระอิ หรือสระอุได้ในกรณี ดิ์ หรือ ธุ์


### ตรวจจับไม้หันอากศ (สระอะเปลี่ยนรูป) ที่ไม่มีตัวสะกด
```
(?:ั)(?![่้๊๋]?[ก-ฮ](?![ะาิีึืุูำั็่้๊๋์อ]|ว[ก-ฮ]))
```
***จับกรณีที่***
- ไม้หันอากาศที่ไม่มีตัวสะกด

### สระอือต้องมีตัวสะกด
```
(?:ื)(?![่้๊๋]?[ก-ฮ](?![ะาิีึืุูำั็่้๊๋์]))
```
***จับกรณีที่***
- จับสระอืที่ไม่มีตัวสะกด ไม่ตรวจสระออหรืออัวลดรูป

### สระแอะลดรูป
```
แ[ก-ฮ]็(?![ก-ฮ](?![ะาิีึืุูำั็่้๊๋์]))
```
***จับกรณีที่***
- สระแอะลดรูป (ไม้ไต่คู้) ไม่สามารถมีวรรณยุกต์ได้
- สระแอะไม่มีตัวสะกด ไม่ตรวจสระออหรืออัวลดรูป

### ตรวจจับวรรณยุกต์ที่ไม่มีพยัญชนะต้น
```
(?<![ก-ฮ][ิีืึุูั]?)[่้๊๋]
```
***จับกรณีที่***
- วรรณยุกต์ไม่มีพยัญชนะนำหน้า
- วรรณยุกต์ที่ถ้ามีสระบนล่างไม่อยู่ก่อนวรรณยุกต์

### ตรวจจับสระหน้าที่ไม่มีพยัญชนะตามท้าย
```
[เแไโใ](?![ก-ฮ])
```
***จับกรณีที่***
- สระหน้าไม่ตามด้วยพยัญชนะ

### ตรวจจับสระนำหน้าที่ไม่ควรสระอื่น
```
(?:[ไใ][ก-ฮ][่้๊๋]?[ะาิีึืำั็์]|[โ][ก-ฮ][่้๊๋]?[าิีึืำั็์]|แ[ก-ฮ][่้๊๋]?[าิีืึุูำั์]|เ[ก-ฮ][่้๊๋]?[ึุูำั์])
```
- สระไอโอตามด้วยพยัญชนะที่มีสระอาอิอีอึอือุอูอำไม้หันอากาศไม้ไต่คู้การันต์
- สระแอตามด้วยพยัญชนะที่ไม่มีสระอาอิอีอึอืออุอูอำไม้หันอากาศการันต์
- สระเอตามด้วยพยัญชนะที่ไม่มีสระอึอุอูอำไม้หันอากาศการันต์


### ตรวจจับการใช้สระอะที่เป็นสระอะและสระเอาะโอะเออะเอะแอะที่ไม่มีพยัญชนะข้างหน้า
```
(?<!เ[ก-ฮ]{1,2}[่้๊๋]?[าอ]|[โเแ]?[ก-ฮ]{1,2}[่้๊๋]?)ะ
```
***จับกรณีที่***
- สระอะไม่มีพยัญชนะนำหน้า
- สระอะและข้างหน้าไม่ใช่รูปแบบสระเอาะ
- สระอะและข้างหน้าไม่ใช่รูปแบบสระโอะ
- สระอะและข้างหน้าไม่ใช่รูปแบบสระเออะ
- สระอะและข้างหน้าไม่ใช่รูปแบบสระแอะ
- สระอะและข้างหน้าไม่ใช่รูปแบบสระเอะ
### ตรวจจับการใช้สระอาที่เป็นสระอาและสระเอาที่ไม่มีพยัญชนะข้างหน้า
```
(?<!เ?[ก-ฮ]{1,2}[่้๊๋]?)า
```
***จับกรณีที่***
- สระอาไม่มีพยัญชนะนำ
- สระอาไม่อยู่ในรูปแบบสระเอา

### สระเอสระเอียและสระเอือสระเออ
```
เ(?![ก-ฮ][่้๊๋]?อ?)|เ[ก-ฮ](?:ี(?![่้๊๋]?ย)|ื(?![่้๊๋]?อ))
```
***จับกรณีที่***
- สระเอแล้วไม่มีพยัญชนะตาม
- สระเอแล้วไม่ใช่สระเอีย 
- สระเอแล้วไม่ใช่สระเอือ

### สระอำ
```
(?:[แเโไใ][ก-ฮ]|[ก-ฮ][ิีึืุูาะ])(?:ำ)|(?:ำ)[่้๊๋]
```
***จับกรณีที่***
- พยัญชนะที่มีสระอำมีสระอื่นร่วมอยู่ด้วย
- สระอำที่วรรณยุกต์อยู่หลัวสระอำ

### มีวรรณยุกต์ไม่มีสระ

# ตรวจจับสระลดรูปสระเปลี่ยนรูป ตรวจสอบพยัญชนะและตัวสะกดในภาษาไทย

การตรวจสอบตัวสะกดในภาษาไทยตรวจสอบได้ยากเพราะภาษาไทยเป็นภาษาที่เขียนติด ๆ กันไม่มีการแบ่งคำอย่างชัดเจนทำให้การอ่านภาษาไทยผู้อ่านต้องใช้ความหมายของคำในการตัดสินการอ่านแบ่งคำตามความเหมาะสมเช่นคำว่า "ตากลม" อาจอ่านเป็น "ตาก-ลม" ก็ได้ หรืออ่านเป็น "ตา-กลม"ก็ได้ 

ดังนั้นการเขียน Regex เพื่อทำการตรวจสอบอาจช่วยได้ระดับหนึ่ง อ่าจมีผิดบ้างถูกบ้าง แต่ก็ถือว่าเป็นเครื่องมือที่ใช้ช่วยเหลือในการตรวจสอบเพิ่มเติมได้ 80% ของความเป็นไปใด้ก็แล้วกันนะครับ

# ตรวจจับคำยืมคำต่างชาติ

วิเคราะห์ลักษณะคำต่างชาติ คำยืม

คอมพิวเตอร์ กราฟ สเกิร์ท ชีส ซัดส์ ดัชท์ สวิทช์ เซ็กซ์ วัตต์ เสปิร์ม สตาร์ แบลคลิสต์ ไวท์ลิสต์

## วิเคราะห์ตามกฏภาษาไทยเพิ่มเติม

**ตัวการันต์กับเครื่องหมายทัณฑฆาตบังคับต้องมีตัวสะกดตาม**

ไม้หันอากาศตามด้วยตัวสะกดทันทีจึงตามด้วยตัวการันต์
สระเออเปลี่ยนรูปต้องใส่ตัวสะกดหลังตัวการันต์
ไม้หันอากาศตามด้วยตัวสะกดทันทีจึงตามด้วยตัวการันต์ (ไม่บังคับมีตัวการันต์ข้างหลัง)
สระออต้องใส่ตัวการันต์ก่อนแล้วตามด้วยตัวสะกด(ไม่บังคับต้องมีตัวสะกด)
สระอี "ไม่บังคับ" ตัวสะกด์แต่ต้องใส่การันต์หลังตัวสะกด

ดังนั้นตามลอจิกแล้วต้องจับที่ผิดไวยากรณ์จึงจะทำเพียงสามสระที่หลักชันเจนคือ **สระเออสระเอะสระอะ(เปลี่ยนรูป)** ที่บังคับตัวสะกดตายตัว

### คำฝรั่งจะมีอักษรนำเพิ่ม
```

```
