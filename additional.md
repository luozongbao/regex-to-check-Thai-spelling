# ตรวจจับเพิ่มเติมพิเศษ

เนื้อหาที่จะจับเพิ่มใน Regex นี้ยากขึ้นไปอีกและไม่คิดว่าจะทำได้ และหรือไม่ควรใส่เข้าไปใน Regex อันมาตรฐานที่ทำไว้ก่อนหน้าเพราะจะทำให้ผลลัพธ์ผิดไปหมดเลยได้

## ข้อความ
กาแฟร้อนๆ มีกลิ่นหอมฟุ้งไปทั่วห้อง เด็ก ๆ กล้อนผมก็วิ่งเล่นอย่างสนุกสนาน เพียงผู้หญิงคนนั้นเดินผ่านไปอย่างรวดเร็ว ใบไม้ร่วงหล่นลงมาเป็นจำนวนมาก ฉันชอบกินผลไม้สดๆ โดยเฉพาะแตงโม สมุทรสาครเป็นจังหวัดที่อยู่ใกล้กรุงเทพฯ เขาแต่งกลอนแล้วพูดว่า "ไม่เป็นไร" แล้วยิ้มให้อย่างแกล้วกล้า นักเรียนทุกคนต้องใส่เครื่องแบบ ดอกไม้บานสะพรั่งในสวน ฤดูฝนปีนี้มีน้ำมาก ญาติๆ มาเยี่ยมเยือนบ้านเป็นประจำ ธรรมชาติสวยงามมาก ฌานสมาธิช่วยให้จิตใจสงบ ปลาทูทอดกรอบอร่อยมาก แม่ซื้อของมาเต็มตะกร้า พ่อขับรถไปทำงานทุกวัน ไก่ตัวนั้นขันดังมาก เธอใจดีมากๆ เลย ชาวบ้านชุมชนไม่สนใจความละเอียดละออ เคล้าความงมงาย มะขามเปรี้ยวมะเขือหวานอร่อยมาก อยากตื่นเต้นดีใจจริงๆ ถ้าเทียบความดั้งเดิมทำไมคนแซ่ตั้งดี เขาว่าอย่าอยู่อย่างอยาก อย่ามาชวนทะเลาะได้ไหม เอาอะไรมาอีกเนาะโมเมไป แบบว่าอำนาจใหญ่โต แบบนี้นั่นนี่โน่น กล้าเกล้ามาดู เงื่อนไขเหล่านี้ เรื่องราวเรื่อยเปื่อย เสื้อผ้าสวยงาม เหมือนเดิมทุกอย่าง คือแกงเผ็ดแกงจืด ใจเย็นใจร้อน เตียงนอนเตาไฟ เกลือป่นพริกไทย เตาไฟลั่นดังเปรี๊ยะ ๆ เปลือกหอยกระเทาะออกมา เหาะเหินเดินอากาศ โบยบินไปในอากาศทุกอย่างโบ๋เบ๋ มาเราจะไปเที่ยวหัวหิน เมากันให้เละเทะและแทะไปเลย สักครู่นึงก็ได้เนื้อหาตุ๊ดตู่ปู่บู้บี้ละ ดีไม่ดีก็ลองดู ก็โน๊ะ หาโอกาสคุยกันหน่อยกับพวกเกรียนคีย์บอร์ด ลำนำลำน้ำแคว เทพีเพรียงเดินเพลิดเพลินเหลือคนาเพียงใจ ดับเบิ้ลเปิ้น กล้องถ่ายกล้วยล้มส้มใหญ่ไปไม่ได้ใกล้ ๆ หรือไกลก็ได้

กาาแฟร้้อนๆ มีกลิ่ินหอมฟุ้งไปทั่ัวห้อง เด็็ก ๆ กล้อนผมก่็วิิ่งเล่่นอย่างสนุ่กสนาน เพีผู้หญิิงคนนัันเดินผ่านไปอย่างรวดเร็ว ใบไมัร่วงหล่นลงมาเป็นจำนวนมาก ฉันชอบกินผลไม้สดๆ โโดยเฉพาะแตงโม สมุทรสาครเป็นจังหวัดที่อยู่ใกล้กรุงเทพฯฯ เขาแต่งกลอนพูดว่า "ไม่เป็ไร" แล้วยิ้้มให้อย่งแกล้วกล้า นักเรียนทุกคนต้องใใส่เครื่องแบบ ดอกไม้บานสะพรั่งในืสวน ฤดูฝนปีนี้มีน้ำมาก ญาติๆ มาเยี่อมเยืยนบ้านเป็นประจำ ธรรมชาติสวยงามมาก ฌานสมาธิช่วยให้จิตใจสงบ ปลาทูทอดกรอบอร่่อยมาก แม่ซื้อของมาเต็่มตะกร้า พ่อขับรถไปทำงานทุูกวัน ไก่ตัวนั้นขันดังมาก เธอใจดีมาาากๆๆ เลย ชาวบ้านชุุมชนไไม่สนใจความละเอียดละออ เคล้้าความงมงาย มะะขามเปร้ียวมะเขือหวานอร่อยมาก อยากตื่นเต้้นดีใจจริงๆ ถ้าเทียบความดั้งเดิมเทำไมคนแซ่ตั้งดี เขาว่าอย่าอยู่อย่างอยาก อย่ามาชวนทะเลาะได้ไหม เอาอะไรมาอีกเนาะโมเมีไป แบบว่าอ่ำนาจใหญ่โต แบบนี้นั่นนี่โนีน กล้าเกล้ามาดู เงื่อนไขเหล่านี้ เรื่องราวเรื่อยเปื่อย เสื้อผ้าสวยงาม เหมือนเดิมทุกอย่าง คืแกงเผ็ดแกงจืด ใจเย็นใจร้อน เตียงนอนเตาไฟ เกลือป่นพริกไทย เตาไฟลั่นดังเปรี๊ยะ ๆ เปลือกหอยกระทาะออกมา เห็าะเหินเดินอากาศ โบ็ยบินไปในอากาศทุกอย่างโบ๋๋เบ๋่ มาเราจะไปเที่ยวหัวหิน เมากันให้เละเทะและแทะไปเลย สักครู่นึงก็ได้เนื้อหาตุ๊ดตู่ปุีบู้บี้ละ ดีไม่ดีก็ลองดู ก็่โน๊าะ หาโอาคุ้ยกัันหน่อย กับพวกเกรียนคีย์์บอร์ด  แลำนีำลำน้ำแคว เทพีเพรียงเดิเพลิดเพลินเหลืคนาเพีใจ ดับเบิ้ลเปิ้น กล้ถ่ายกล้ล้มส้มหญ่ไปไม่ได้ใกล้ ๆ หรือกลก็ได้

## Regex ตามกฏภาษาไทยเพิ่มเติม

### มีวรรณยุกต์ไม่มีสระ
```
(?<![เแไใโ][ก-ฮ]?)[ก-ฮ](?![ิีึืุู])[่้๊๋](?![ะำา]|[อว]?[กขฆคดตฎฏจชฒถธฑซศษสบปผพฟภงนลรฬญณมย])
```
1. ถ้ามีพยัญชนะและวรรณยุกต์ไม่มีสระหน้าแต่มีสระบนล่าง:ไม่จับ ถ้าไม่มีสระบนล่าง... เชคต่อไป
2. ถ้ามีพยัญชนะและวรรณยุกต์ไม่มีสระบนล่างแต่มีสระหน้า:ไม่จับ ถ้าไม่มีสระหน้าไม่มีสระบนล่าง... เชคต่อไป
3. ถ้ามีพยัญชนะและวรรณยุกต์ไม่มีสระบนล่างไม่มีสระหน้าแต่มี สระออ สระอัว เช่นกล่อม กล้วย ไม่มจับ 
4. อื่น ๆ จับ

### สระอัวต้องมีตัวสะกด
```
[ก-ฮ]{1,2}[่้๊๋]?[ว](?![กขฆคดตฎฏจชฒถธฑซศษสบปผพฟภงนลรฬญณมย])
```
### ตรวจจับอักษรนำอย่าอยู่อย่างอยาก
```
((?<!(?<![เแไโใ])[ก-ฮ]{1,2}[่้๊๋]?|[ก-ฮ]ื[่้๊๋]?)อย)(?!่า|ู่|่าง|าก)
```
จับกรณีที่
- "อย" ไม่ใช่ส่วนหนึ่งของสระออและสระเอือแต่ไม่ใช่ "อย่า" "อยู่" "อย่าง" "อยาก"
