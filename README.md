# 🔍 Regex ตรวจมาตรฐานการสะกดภาษาไทย

โปรเจกต์สำหรับรวบรวม Regular Expression ที่ใช้ตรวจสอบรูปแบบการสะกดคำภาษาไทยขั้นพื้นฐาน (โครงสร้างสระ–วรรณยุกต์–ตัวสะกด) ตามกฎไวยากรณ์ไทยเบื้องต้น

> ⚠️ **หมายเหตุ**: Regex ชุดนี้ตรวจสอบ **รูปแบบการสะกด** เท่านั้น ไม่ได้ตรวจสอบความหมายหรือความถูกต้องของคำในพจนานุกรม แต่จะช่วยจับรูปเขียนที่ "ไม่น่าเป็นไปได้" ตามกฎการสะกดภาษาไทย

---

## 💡 แนวคิด

อาศัยลักษณะเฉพาะของระบบการเขียนภาษาไทย เช่น:
- 📝 สระหน้า (เ แ โ ไ ใ) ต้องมีพยัญชนะตาม
- 🔤 สระบน/ล่าง ต้องมีพยัญชนะรองรับ
- ✍️ ไม้หันอากาศและวรรณยุกต์ต้องวางตำแหน่งถูกต้อง
- ❌ ห้ามมีสระ/วรรณยุกต์/ทัณฑฆาตซ้ำซ้อน
- 🔄 ลำดับของสระ–วรรณยุกต์ต้องไม่สลับกัน

จากนั้นเขียนเป็น Regular Expression เพื่อ "จับตำแหน่งที่ผิดรูป" ในข้อความภาษาไทย

---

## 🎯 ขอบเขตการใช้งาน

### ✅ สิ่งที่ตรวจได้
- การซ้ำซ้อนของสระและวรรณยุกต์
- ลำดับของสระและวรรณยุกต์ที่ผิด
- การขาดพยัญชนะรองรับสระ
- โครงสร้างสระประสมที่ผิด
- การใช้ไม้หันอากาศและทัณฑฆาตที่ไม่ถูกต้อง

### ❌ ข้อจำกัด
- ไม่สามารถตรวจสอบความถูกต้องของคำตามพจนานุกรม
- ไม่ตรวจสอบบริบทหรือความหมายของประโยค
- อาจมี false positive/negative กับคำยืมภาษาต่างประเทศ
- การตรวจตัวสะกดมีข้อจำกัดเนื่องจากภาษาไทยเขียนติดกันไม่มีช่องว่าง

## 📚 องค์ประกอบภาษาไทยที่ตรวจสอบ

### สระ
| ประเภท | ตัวอักษร | ตัวอย่าง |
|--------|---------|---------|
| **สระหน้า** | เ แ โ ไ ใ | เก้อ, แก้ว, โค้ก, ไก่, ใจ |
| **สระบน** | อิ อี อึ อื อั อ็ | กิน, ดี, ถึก, ลืม, กัด, กระดิ่ง |
| **สระล่าง** | อุ อู | กุ้ง, ดู |
| **สระหลัง** | า ะ ำ | กา, ระ, น้ำ |

### อักษรควบคุม
| ประเภท | ตัวอักษร | ตัวอย่าง |
|--------|---------|---------|
| **วรรณยุกต์** | อ่ อ้ อ๊ อ๋ | ค่า, ค้า, ค๊า, ค๋า |
| **ทัณฑฆาต** | อ์ | กรุงเทพฯ, จนท. |

## 🔧 กฎการตรวจสอบ

รายละเอียดครบถ้วนของ Regex แต่ละรูปแบบสามารถดูได้ที่ [regex.md](regex.md)

### 1️⃣ ตรวจสระและเครื่องหมายซ้ำซ้อน
```regex
[เแโไใ]{2,}         # สระหน้าซ้ำซ้อน: กาาแฟ
[ิีึืูุั็]{2,}        # สระบน/ล่างซ้ำ: กลิ่ิน
[่้๊๋]{2,}           # วรรณยุกต์ซ้ำ: ร้้อน
[์]{2,}             # ทัณฑฆาตซ้ำ
```

### 2️⃣ ลำดับที่ผิด
```regex
[่้๊๋][ิีึืุูั็]      # วรรณยุกต์มาก่อนสระบน/ล่าง: ก้ิน (ผิด)
```

### 3️⃣ การผสมสระที่ไม่ถูกต้อง
```regex
[เแโไใ][ิีึืุูั็าะำ]  # สระหน้าตามด้วยสระอื่นทันที: เิดิน
```

### 4️⃣ การขาดพยัญชนะรองรับ
```regex
(?<![ก-ฮ][ิุ]?)(?:์)              # ทัณฑฆาตไม่มีพยัญชนะนำหน้า
(?<![ก-ฮ])(?:ั)                    # ไม้หันอากาศไม่มีพยัญชนะ
(?<![ก-ฮ][ิีืึุูั]?)[่้๊๋]         # วรรณยุกต์ไม่มีพยัญชนะ
[เแไโใ](?![ก-ฮ])                   # สระหน้าไม่มีพยัญชนะตาม
```

### 5️⃣ โครงสร้างสระพิเศษ
```regex
[ไโใ][ก-ฮ][่้๊๋]?[ิีึืุูำั็า์]     # ไ โ ใ ไม่ควรมีสระอื่นร่วม
แ[ก-ฮ][่้๊๋]?[ีืึิุูำัา์]         # แ ไม่ควรมีสระบางตัวร่วม
เ[ก-ฮ][่้๊๋]?[ึุูำั์]             # เ ไม่ควรมีสระบางตัวร่วม
```

## 💻 วิธีใช้งาน

### ตัวอย่างใน JavaScript

```javascript

// เจอรูปผิดที่ตำแหน่ง 1 => "าา"
// เจอรูปผิดที่ตำแหน่ง 6 => "้้"
// เจอรูปผิดที่ตำแหน่ง 15 => "ิี"
// เจอรูปผิดที่ตำแหน่ง 31 => "ั่"
```

### ตัวอย่างใน Python

```python
import re

# เลือก Regex เวอร์ชันมาตรฐาน
pattern = r'[เแโไใ]{2,}|[ิีึืูุั็]{2,}|[่้๊๋]{2,}|[์]{2,}|[่้๊๋][ิีึืุูั็]|[เแโไใ][ิีึืุูั็าะำ]|(?<![ก-ฮ][ิุ]?)(?:์)|(?<![ก-ฮ])(?:ั)|(?:ั)(?![่้๊๋]?(?:(?![ผฝหอฮฉฌรฃฅ])[ก-ฮ]))|(?<![ก-ฮ][ิีืึุูั]?)[่้๊๋]|[เแไโใ](?![ก-ฮ])|[ไโใ][ก-ฮ][่้๊๋]?[ิีึืุูำั็า์]|แ[ก-ฮ][่้๊๋]?[ีืึิุูำัา์]|เ[ก-ฮ][่้๊๋]?[ึุูำั์]|แ[ก-ฮ]็(?![ก-ฮ])|(?<![ก-ฮ][่้๊๋]?)[ิีึืุูำำ]|(?<!เ[ก-ฮ]{1,2}[่้๊๋]?[าอ]|[โเแ]?[ก-ฮ]{1,2}[่้๊๋]?)ะ|(?<!เ?[ก-ฮ]{1,2}[่้๊๋]?)า|เ(?![ก-ฮ])|เ[ก-ฮ](?:ี(?![่้๊๋]?ย[ก-ฮ]?)|ื(?![่้๊๋]?อ[ก-ฮ]?))|[ื](?![่้๊๋]?[ก-ฮ](?![ะาิีึืุูำั็่้๊๋]))|(?:[แเโไใ][ก-ฮ]|[ก-ฮ][ิีึืุูาะ])ำ'

text = "กาาแฟร้้อนๆ มีกลิ่ินหอมฟุ้งไปทั่ัวห้อง"

# ค้นหาข้อผิดพลาดทั้งหมด
matches = re.finditer(pattern, text)
for match in matches:
    print(f'เจอรูปผิดที่ตำแหน่ง {match.start()} => "{match.group()}"')
```

## ⚠️ ข้อควรระวังและข้อจำกัด

### ข้อควรระวัง
1. **ไม่ตรวจสอบความหมาย**: Regex ชุดนี้ตรวจเฉพาะ "รูปแบบ" ไม่ได้ดูว่าคำมีความหมายหรือมีอยู่ในพจนานุกรมจริง
2. **False Positive/Negative**: อาจมีคำที่เขียนถูกแต่ถูกจับว่าผิด หรือเขียนผิดแต่ไม่ถูกจับ
3. **คำภาษาต่างประเทศ**: คำยืมจากภาษาต่างประเทศที่เขียนด้วยอักษรไทยอาจถูกจับว่าผิด
4. **ตัวสะกด**: การตรวจตัวสะกดมีข้อจำกัดเพราะภาษาไทยไม่มีช่องว่างแบ่งคำ ทำให้ยากต่อการระบุขอบเขตคำ

### แนวทางการใช้งาน
- ✅ ใช้เป็นเครื่องมือช่วยกลั่นกรองเบื้องต้น

- ✅ ควรมีมนุษย์ตรวจสอบอีกครั้งเพื่อความแม่นยำ
